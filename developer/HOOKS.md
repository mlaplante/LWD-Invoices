- **'invoice_all_unpaid'**
All unpaid invoices

- **'invoice_sent_but_unpaid'**
List of invoices that are marked as sent but unpaid

- **'invoice_all_overdue'**
Get all overdue invoices

- **'invoice_get_all_unseen'**
Get list of all invoices that are unseen

- **'invoice_days_overdue'**
Count of days an invoice is overdue

- **'invoice_next_reoccurrence_date'**
Returns the next date for invoice recurrence

- **'invoice_unique_id_generated'**
Returns the generated unique ID used for the url of the invoice

- **'before_invoice_number_generated'**
Returns the count of a custom invoice number before the default invoice number is generated.

- **'invoice_number_generated'**
Returns the generated invoice number
